# instagrab
Frontend grabber from Instagram's users

Dependencies 'HTTParty'

```$ sudo gem install httparty```

**Usage**

```$ ruby instagrab.rb [profile-name-without-@]```

**Example**

```$ ruby instagrab.rb jmozac```

**Output**
```
Username @jmozac
  ID 		: 30611932
  Followers	: 954
  Following	: 53
  Post 		: 164
```
